# web_banhmi
Used Technologies:
    PHP, 
    MySQL, 
    HTML, CSS, JavaScript, 
    Bootstrap, 
    SQL Query

About Project:

User Management: Users can register, log in, edit their profile, and change their password.
Product Browsing: Users can view various bread products, see detailed information, and search based on criteria.
Shopping Cart: Users can add products to the cart, update quantities, remove items, and proceed to checkout.
Order History: Users can review past purchases
Admin Panel: Administrators can manage products, users, and orders.
Supporting Pages: The website includes news and contact pages for a better user experience.
